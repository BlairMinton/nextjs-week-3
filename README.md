# Group Activity: Dynamic Routes with DummyJSON
In this activity you will work together in a group of **three students** to build **three dynamic routes** in a Next.js application.

Each route should display **a single item of data fetched from the DummyJSON API**:
[https://dummyjson.com/docs](https://dummyjson.com/docs)

Ensure you read the documentation to understand what kind of data is returned from the API for each route. 

---

## Instructions
1. Create a fork of this repository
2. Clone your forked version of this repository to your computer
2. As a group, determine who will cover what route
3. Create the route that you have been designated
4. Create the page.tsx file and implement fetching from the API
5. Display 3 pieces of information on the page from the data returned from the API 
6. Stage, commit, and push the changes to your fork
7. Create a pull request on _this_ repository merging the changes from your fork to this repositories `master` branch

All three students should:

- Contribute to writing code
- Discuss how the dynamic route works
- Ensure everyone understands how the data is retrieved
- Test each route together

## Routes
As a team determine who should cover route 1, route 2 and route 3.

### Route 1

```
/products/[id]
```

The page should display 3 pieces of information about **one product**.

---

### Route 2

```
/users/[id]
```

The page should display 3 pieces of information about **one user**.

---

### Route 3

```
/posts/[id]
```

The page should display 3 pieces of information about **one post**.
